O que é?

  • Classes são templates para criação de objetos
    • Classes podem facilitar a criação de diversos objetos com a mesma estrutura

  • Pode conter atributos e métodos
    • Atributos: são as características dessa classe
    • Já os métodos são as funções/li>

Quando usar

  • Quando tiver um conjunto de dados que pode ter o mesmo padrão
    • Ex: Dados de pessoas, dados de pedidos

  • Quando precisar encapsular diversas funções de um recurso
    • Ex: Conta bancária
